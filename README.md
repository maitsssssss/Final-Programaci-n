# Final-Programaci-n
App Health Track es una aplicación de seguimiento de salud que te ayuda a registrar tu peso, medidas corporales, objetivos saludables y más.

Instalación
Clona este repositorio en tu máquina local:
```bash
git clone https://github.com/tu_usuario/AppHealthTrack.git

Uso
Ejecuta el script main.py para iniciar la aplicación.
python main.py
La aplicación te guiará a través de las opciones disponibles. Puedes crear nuevos usuarios, ingresar nuevos pesos y medidas, ver ideas de comida, ejercicios recomendados, tips saludables, y más.

Ejemplos de uso

Crear un Nuevo Usuario
Al seleccionar la opción "Crear nuevo usuario", la aplicación te pedirá información como nombre, peso inicial, altura, medidas corporales y objetivos saludables. Después de proporcionar esta información, se creará un nuevo usuario en la aplicación.

Ingresar Nuevo Peso y Medidas
Puedes actualizar el peso y las medidas de un usuario existente seleccionando la opción "Ingresar nuevo peso y medidas de usuario existente". La aplicación te guiará a través del proceso de ingreso de esta información.

Ver Visualmente el Progreso de un Usuario
Selecciona la opción "Mostrar progreso" en el menú principal e ingresa el nombre del usuario del que deseas ver el progreso. La aplicación generará un gráfico que muestra el historial de pesos del usuario a lo largo del tiempo, lo que te permitirá visualizar fácilmente su progreso. Puedes observar si el usuario está progresando hacia sus objetivos de pérdida o ganancia de peso y ajustar su plan de salud en consecuencia.

Ver Tips Saludables, Ideas de Comida o Ejercicios Recomendados 

Selecciona la opción "Mostrar tips saludables", "Mostrar ideas de comida" o "Mostrar ejercicios recomendados" en el menú principal para acceder a una lista de consejos y recomendaciones para llevar un estilo de vida saludable, diferentes recetas o distitnos ejercicios.
